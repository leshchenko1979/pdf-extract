# Security

## Reporting a vulnerability

Please report security issues **privately** so they can be addressed before public disclosure.

- Prefer **[GitHub Security Advisories](https://github.com/leshchenko/pdf-extract/security/advisories/new)** for this repository (if the repo is hosted under that path).
- If that is not available, open a **private** report via the contact options on the maintainer’s GitHub profile.

Please include steps to reproduce, affected versions or commits, and impact if known.

## Scope

This service fetches PDFs from user-supplied URLs and processes uploads. Review [README.md](README.md) (URL fetch safety, size limits) when deploying publicly.
